wf-xano examples

Self-contained demos with a mocked Xano backend — view source on any page to see the exact markup.

Brand opportunities feed

The full grammar on one page: template binds with wf-xano-format, status pills via wf-xano-if, root-anchor links, a select filter, numbered pagination, outside-root counts via wf-xano-instance, and URL sync (wf-xano-url-sync — paginate, filter, then reload).

Filters, search & sort

Debounced search (wf-xano-search), a checkbox group combining values via wf-xano-value, a renamed sort param, and the is-wf-xano-loading state class dimming the list during requests — with every control living outside the list wrapper.
