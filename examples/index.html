Brand opportunities feed

The full grammar on one page: template binds with wf-xano-format, status pills via wf-xano-if, root-anchor links, a select filter, numbered pagination, outside-root counts via wf-xano-instance, URL sync (wf-xano-url-sync — paginate, filter, then reload), and a pessimistic wf-xano-action Archive button (v0.21).

Brand view — real-world case

Model of The Starters' /opportunities-brands-view migration: the same card design rendered from brand/opportunities/list instead of a search index. Click Post opportunity to see the fix that motivated the library — a just-posted job is visible immediately (no indexing lag), with the endpoint's status input driving the filter.

Filters, search & sort

Debounced search (wf-xano-search), a checkbox group combining values via wf-xano-value, a renamed sort param, and the is-wf-xano-loading state class dimming the list during requests — with every control living outside the list wrapper.

Authenticated Memberstack list

A complete mocked token trade: Memberstack session cookie → POST body → Xano bearer token → member-scoped list, with loader, empty, error and accessible request states.
